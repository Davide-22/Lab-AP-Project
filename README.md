# Run using `docker-compose`

### Prerequisites

1. Docker
2. Docker-Compose

### Run
```bash
docker-compose up --build -d
```

You can check the containers with Docker Desktop or by terminal
```bash
docker ps
```

The application is running on http://localhost/
